Variáveis

Variáveis são espaços na memoria, como gavetas que pode guardar informações dentro delas, existem 3 tipos var, let, const (constante não muda o valor dentro). São declaradas da seguinte forma:

let nome = valor;

var nome = valor; ==> Não é mais usada

const nome = valor; ==> Constante o valor não pode ser mudado com o decorrer da execução do código

Existem vários tipos de valores (simples):
  1. Number (valor númerico)
  2. String ('valor de caractere')
  3. Boolean (representado por true ou false)
  4. Null (ausência de valor)
